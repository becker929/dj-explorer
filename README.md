# dj-explorer

A web app for exploring DJ mixes from SoundCloud, made in Flask with Flask-SocketIO. 



## How to run



$ python3 main.py


This will run the Flask server, which can the be accessed via the browser at http://0.0.0.0:80/mix-explorer .
